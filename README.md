# LessonFlutterDemo

## 摘要
> 这是一个Flutter的demo，将基本的UI操作和数据传递，自定义view进行了简单的操作。
> 这个项目是根据 [Flutter实践](https://book.flutterchina.club/) 进行学习编辑的，如果有什么明白的可以点击链接查看更详细的内容。
> 项目还不是非常完善，还有非常多的需要学习的内容，后续会继续学习和维护，欢迎star。
